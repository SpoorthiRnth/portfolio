Twelve agents govern every stage of supply chain data flow across a 4-zone Medallion architecture. Found $204,950 in financial exposure and $7,200 in invoice overbilling in a single pipeline run.

App Management — Autonomous Incident Resolution
Four AI agents triage, investigate, and resolve operational incidents autonomously. High-risk changes — config updates, rollbacks, status writes — pause for human approval before anything executes.
Incident raised → Orchestrator → SQL Agent → Resolver → Human approval → Ticket closed
The proposal tools are structurally incapable of writing to the database — the only path to a write is through execute_approved_proposal, which requires an approved_by value in the audit log. This cannot be bypassed by an agent. Human-in-the-loop is architecture, not a safety patch.
14 resolution playbooks loaded at runtime — one per incident type. Adding a new incident type means adding a playbook entry, no code changes. Every approved change produces an immutable audit log with old value, new value, operator name, and timestamp.
AutoGen GPT-4o Multi-Agent Human-in-the-Loop Python SQLite
